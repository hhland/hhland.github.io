Data requirements for one-way ANOVA  

What kind of data is required?

You must have continuous data that are divided into two or more groups. For example:
  • Strength measurements for samples of four different brands of wire
  • Customer satisfaction ratings for six different locations of a hotel chain
  • Length of parts produced on three different manufacturing lines
This data can be organized as either of the following:
  • Response data are in single column for all factor levels
  • Response data are in a separate column for each factor level

Your samples must be independentÂ—that is, the observed values in one sample do not depend on the observed values of another sample. Unlike a t-test, ANOVA can compare more than two groups.

What should my worksheet look like?

Response data are in single column for all factor levels

The continuous response data for all of the groups are in a single column (in this example, Strength) with a column of numeric, text, or date/time data that identifies the group of each observation (in this example, Additive).
