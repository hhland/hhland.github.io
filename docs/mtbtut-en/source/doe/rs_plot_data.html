Data requirements for response surface design plots.  

What kind of data is required?

Once you have analyzed your design you are ready to create factorial, contour, and response surface plots.

You must fit a response surface design model with two or more continuous variables before you can generate a contour plot. Contour plot does not use the data in the worksheet. Instead, Minitab looks in the worksheet for a stored model to obtain the necessary information.

Minitab automatically stores the most recent model that you fit for each response variable.

What should my worksheet look like?

When you create a design, Minitab stores the standard order, run order, center point indicator, block assignment, and each factor in separate columns. Each row in the worksheet contains data corresponding to one run of your experiment. The exact makeup of your worksheet will vary depending on your design choices, factor information, and responses.
