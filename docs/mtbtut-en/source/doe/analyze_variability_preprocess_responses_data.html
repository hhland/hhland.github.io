Data requirements for Analyze Variability and Preprocess Responses  

What kind of data is required?

You can use Create Factorial Design to assemble the required data for you, or you can use Define Custom Factorial Design to create a design from data you already have in the worksheet.

After you conduct the experiment, enter the response data for each run into the worksheet. Your response data must include at least one of following:
  • Repeat measurements taken during the same experimental run or consecutive runs
  • Replicate measurements taken during identical but distinct experimental runs, which are often randomized
  • Precalculated standard deviations for the responses

Before you can use Analyze Variability, you must use Pre-Process Responses for Analyze Variability to calculate the standard deviation within each experimental condition or to identify precalculated standard deviations.

What should my worksheet look like?

When you create a design, Minitab stores the standard order, run order, center point indicator, block assignment, and each factor in separate columns. Each row in your worksheet corresponds to one run of your experiment. After you conduct the experiment, enter the response data for each run.

The tables below shows two worksheets after Pre-Process Responses for Analyze Variability is used. The first worksheet uses replicates, and the second worksheet uses repeat measurements.
