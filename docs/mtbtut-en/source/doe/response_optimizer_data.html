Data requirements for response optimizer.  

What kind of data is required?

To use the Response Optimizer, you must fit a model. You can fit a model with seven features in Minitab:
  • Analyze Factorial Design
  • Analyze Response Surface Design
  • Analyze Mixture Design
  • Fit Regression Model
  • Fit General Linear Model
  • Fit Binary Logistic Model
  • Fit Poisson Regression Model

What should my worksheet look like?

When you create a design, Minitab stores the design data in separate columns. Each row in the worksheet contains data corresponding to one run of your experiment. The exact makeup of your worksheet will vary depending on your design type and other choices.
