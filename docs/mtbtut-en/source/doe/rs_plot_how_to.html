Description of how to create response surface design plots using Minitab  

Scenario

A chemical manufacturer wants to maximize the yield of a catalytic reaction. The researchers chose a central composite design to study the effects of time and temperature on the yield of the reaction. Because the data will be collected on two different days, the design contains two blocks.

The researchers fit the model with Analyze Response Surface Design and want to use the stored to model to create a surface plot.

Create a contour plot

Create a contour plot to find the factor settings that provide the response you want.
  1. Open Reaction_model.MTW.
  2. Choose Stat > DOE > Response Surface > Contour Plot.
  3. The default settings are correct so you can click OK.
