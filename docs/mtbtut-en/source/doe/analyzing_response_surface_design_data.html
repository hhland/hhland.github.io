Data requirements for  

What kind of data is required?

You can use Create Response Surface Design to assemble the required data for you, or you can use Define Custom Response Surface Design to create a design from data you already have in the worksheet.

After you conduct the experiment, enter the response data for each run into the worksheet. You can have up to 25 numeric response data columns. Examples of response variables are:
  • Yield of a chemical reaction
  • Time to process an order
  • Strength of plastic

What should my worksheet look like?

When you create a design, Minitab stores the standard order, run order, center point indicator, block assignment, and each factor in separate columns. Each row in the worksheet contains data that corresponds to one run of your experiment. The exact makeup of your worksheet will vary depending on your design choices, factor information, and responses.

If you specify factor level settings, Minitab displays the actual values in the worksheet. Otherwise, Minitab displays the factor levels in coded units: -1 for the low value, 0 for the center point, and 1 for the high value. Axial points are coded as -1.41421 and 1.41421. In either case, you usually want to analyze your design in coded units.
