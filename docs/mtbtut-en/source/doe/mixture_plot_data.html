Data requirements for mixture design plots.  

What kind of data is required?

If you have used Analyze Mixture Design to fit a model, you have the data necessary to create simplex design plots, factorial plots, and contour and surface plots.

What should my worksheet look like?

When you create a design, Minitab stores the standard order, run order, point type, block assignment, and each component and process variable in separate columns. Each row in the worksheet contains data corresponding to one run of your experiment. The exact makeup of your worksheet will vary depending on your design choices, component information, process variables, and other considerations.
