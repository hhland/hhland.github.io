Data requirements for  

What kind of data is required?

You can use Create Mixture Design to assemble the required data for you, or you can use Define Custom Mixture Design to create a design from data you already have in the worksheet.

After you conduct the experiment, enter the response data for each run into the worksheet. You can have up to 25 numeric response data columns. Examples of response variables are:
  • Taste score
  • Yield of a crop
  • Adhesive properties of paint

What should my worksheet look like?

When you create a design, Minitab stores the standard order, run order, point type, block assignment, and each component and process variable in separate columns. Each row in the worksheet contains data that corresponds to one run of your experiment. The exact makeup of your worksheet will vary depending on your design choices, component information, process variables, and other considerations.
