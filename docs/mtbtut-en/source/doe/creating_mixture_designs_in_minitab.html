Description of how to a using Minitab  

Scenario

Researchers in a food laboratory are trying to improve a recipe for cheese fondue. They would like to study how the mixture blend and the serving temperature of the fondue affect the flavor.

The mixture is comprised of three ingredients: two types of Swiss cheese (Emmenthaler and Gruyere) and white wine.

Choose the appropriate design and number of components

The researchers want to conduct an extreme vertices design with 3 components (the cheeses and wine) and one process variable (serving temperature).
  1. Choose Stat > DOE > Mixture > Create Mixture Design.
  2. Under Type of Design, choose Extreme vertices.
  3. From Number of Components, choose 3.
  4. Click Designs. Click OK.

Name components and define bounds

Specify the lower and upper bounds for the components. In this example, past experience has established limits on each ingredient in the overall fondue formulation.
  1. Click Components and complete the table as follows:
    Component Name Lower Upper
    A Emmenthaler 0.2 0.6
    B Gruyere 0 0.3
    C Wine 0.4 0.6
  2. Click OK.

Define process variables

You can define up to 7 process variables, though this example uses only one: serving temperature. You must specify the process variable name, data type, and low and high settings.
  1. Click Process Vars.
  2. Under Process Variables, choose Number and complete the table as follows:
    Process Variable Name Type Low High
    X1 Temperature Numeric 80 90
  3. Click OK in each dialog box.

Interpreting the output
