Description of how to create a P chart using Minitab  

Scenario

A telephone service center wants to assess whether its call answering system is in control. The total number of incoming calls and the number of unanswered calls were recorded for 21 days. Each unanswered call is a defective.

Choose the appropriate analysis

Because defectives are being monitored, you can use either a P or NP chart; however, because the subgroup size (number of incoming calls each day) varies, a P chart is more appropriate.
  1. Open the worksheet TELEPHONE.MTW.
  2. Choose Stat > Control Charts > Attributes Charts > P.

Enter your data

You have one column that contains the defectives data and one column that contains the subgroup sizes. If the subgroup size is constant, you can enter this value instead of a column.
  1. In Variables, enter Unanswered Calls.
  2. In Subgroup sizes, enter P Chart Subgroups.
  3. Click OK.

Specify options

This chart offers many helpful options. You can add labels, split the chart into stages, subset the data used for your chart, and so on. In this case, you are concerned with not missing any possible special causes of variation.
  1. Click P Chart options, and then choose Tests.
  2. Choose Perform all tests for special causes, and then click OK in each dialog box.
