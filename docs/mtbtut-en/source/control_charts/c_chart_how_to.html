Description of how to create a C chart using Minitab  

Scenario

A wallpaper manufacturer wants to assess the stability of its printing process. Samples of 100 feet were taken from 25 wallpaper rolls and the number of blemishes, which include print smears, pattern distortions, and missing ink, were counted.

Choose the appropriate analysis

Both C and U charts display counts of defects (here, the printing blemishes), and both charts are appropriate when sample size is constant (here, 100-foot pieces). In this case, a C chart was chosen out of personal preference.
  1. Open the worksheet WALLPAPER.MTW.
  2. Choose Stat > Control Charts > Attributes Charts > C.

Enter your data

You have a single column of counts; entering more than one column in Variables produces as many C charts.
  1. In Variables, enter Wallpaper.

Specify options

This chart offers many helpful options. You can add useful labels, split the chart into stages, subset the data that is used for your chart, and so on. In this case, you are concerned with not missing any possible special causes of variation.
  1. Click C Chart options, and then choose Tests.
  2. Choose Perform all tests for special causes, and then click OK in each dialog box.
