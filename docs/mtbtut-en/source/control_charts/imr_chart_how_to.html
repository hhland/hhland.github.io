Description of how to create an I-MR chart using Minitab  

Scenario

A liquid detergent company wants to assess whether its process is in control. The company makes the liquid detergent in batches by mixing together a number of ingredients. The quality characteristic of interest is the pH value for each batch. Measurements of the pH value for 25 consecutive batches were taken.

Choose the appropriate analysis

Because the company created the liquid detergent in a batch process and took only one measurement for each batch, you should not place the data in subgroups. Instead, you should analyze the liquid detergent data using an I-MR chart, which considers each measurement to be an independent observation.
  1. Open the worksheet DETERGENT.MTW.
  2. Choose Stat > Control Charts > Variables Charts for Individuals > I-MR.

Enter your data

You have a single column of data; entering more than one column in Variables produces as many I-MR charts.
  1. In Variables, enter pH.

Specify options

This chart offers many helpful options. You can add labels, split the chart into stages, subset the data used for your chart, and so on. In this case, you are concerned with not missing any possible special causes of variation.
  1. Click I-MR Chart options, and then choose Tests.
  2. Choose Perform all tests for special causes, and then click OK in each dialog box.

Interpreting the output

Now what? For guidance on interpreting the results of this analysis, turn to StatGuide.
