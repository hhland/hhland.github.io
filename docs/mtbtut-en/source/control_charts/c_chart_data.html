Data requirements for C chart  

What kind of data is required?

You must have a single column that contains the number of defects for each subgroup, given a consistent subgroup size. For example:
  • Number of surface blemishes (defects) counted on 10 sample panes of glass (subgroup) each day
  • Number of drips (defects) from 4 spray painted truck tailgates (subgroup) collected each hour
  • Number of surface blemishes (defects) counted on 10 sample panes of glass (subgroup) each day
Note You can also use a U chart to monitor defects because C and U charts are interchangeable when the subgroup size is constant. However, you should use a U chart when subgroup sizes vary.
