Data requirements for NP chart  

What kind of data is required?

You must have a single column that contains the number of defectives for each subgroup. If the subgroup size is not constant, you must also have a column that contains the corresponding subgroup sizes. If the subgroup size is constant, you do not need to record it in the worksheet; instead, enter this value in the dialog box. For example:
  • Number of television sets that fail inspection each day (defectives) and number of sets that are inspected (subgroup)
  • Number of emergency room registration forms that lack critical information (defectives) and total number of registration forms submitted
  • Number of grounded aircraft in a delivery company's fleet (defectives) and total number of aircraft it owns
Note You can also use a P chart to monitor defectives because it has identical data requirements. A P chart may be more appropriate when subgroup sizes vary because it maintains a constant centerline.

What should my worksheet look like?

Defectives in one column, subgroup size in another column

Each row contains the size of the subgroup and the number of defectives found within that subgroup.
