Data requirements for U Chart  

What kind of data is required?

You must have a single column that contains the number of defects for each subgroup. If the subgroup size is not constant, you must also have a column of corresponding subgroup sizes. If the subgroup size is constant, you do not need to record it in the worksheet; instead, note this value in the dialog box. For example:

  • Number of surface blemishes counted on a sample of glass (defects) and the number of panes inspected (subgroup)
  • Number of drips from a sample of spray-painted truck tailgates (defects) and the number of parts inspected (subgroup)
Note You can also use a C chart to monitor defects because the two charts are interchangeable when the subgroup size is constant. However, you should use a U chart when subgroup sizes vary.
