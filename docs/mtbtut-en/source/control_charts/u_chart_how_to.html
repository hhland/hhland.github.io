Description of how to create a U Chart using Minitab  

Scenario

A stenographic company wants to assess the quality of its transcription service. They sampled 25 sets of pages and counted the number of typographical errors for each set. Each set had a different number of pages, so the sample size varies. Typographical errors are defects.

Choose the appropriate analysis

Although C and U charts display counts of defects (the typographical errors, in this case), only the U chart is appropriate when sample size varies.
  1. Open the worksheet TRANSCRIPTION.MTW.
  2. Choose Stat > Control Charts > Attributes Charts > U.

Enter your data

You should have a single column of counts and either a column containing the subgroup sizes (if it varies) or an integer (if it is constant). Entering more than one column in Variable produces as many U charts.
  1. In Variables, enter Typos.
  2. In Subgroup sizes, enter Pages.

Specify options

This chart offers many helpful options. You can add labels, split the chart into stages, subset the data that is used for your chart, and so on. In this case, you are concerned with not missing any possible special causes of variation.
  1. Click U Chart options, and then choose Tests.
  2. Choose Perform all tests for special causes, and then click OK in each dialog box.
