Description of how to perform 1 Proportion using Minitab  

Scenario

A direct-mail firm chooses 1000 households at random to receive advertisements for a new laundry detergent. Of the 1000 households that receive the mailing, 87 purchase the detergent. The firm wants to determine if this success rate is different from their average of 6.5%.

Choose the appropriate analysis

The firm wants to compare their proportion to a reference value. The 1 Proportion test is appropriate for this situation.
  1. Choose Stat > Basic Statistics > 1 Proportion.

Enter your data

The firm recorded values for the number of trials and number of events, so they provide summarized data instead of using a worksheet.
  1. Choose Summarized data.
  2. In Number of events, type 87.
  3. In Number of trials, type 1000.

Perform a hypothesis test

The firm wants to test whether their proportion differs from the national average of 6.5%.
  1. Check Perform hypothesis test.
  2. In Hypothesized proportion, type 0.065.
  3. Click OK.

Specify options

This analysis requires no further modifications, but other options are available. Use the Options subdialog box to change the confidence level, to request a one-sided test, or to specify a normal approximation for the test and confidence interval.
  • (no action required)
