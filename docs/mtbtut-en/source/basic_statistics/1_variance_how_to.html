Description of how to perform 1 Variance using Minitab  

Scenario

The manager of a lumber yard wants to analyze the performance of the saw mills. One mill is designed to produce beams that are exactly 100 cm long. The manager decides to analyze the variance of these lengths to better understand the precision of the equipment. The manager takes a sample of 50 beams, measures their length in centimeters, and analyzes the variance using the 1 Variance test.

Choose the appropriate analysis

The manager wants to test the variance of a single group. A 1 variance test is appropriate for this situation.
  1. Open the worksheet SAWMILL.MTW.
  2. Choose Stat > Basic Statistics > 1 Variance.

Enter your data

This worksheet contains a single column of sample data.
  1. Choose One or more samples, each in a column and enter Length.
  2. Click OK.

Perform a hypothesis test

For this analysis, only a confidence interval for the variance is necessary, but you could use this option to test whether the variance equals a target value.
  • (no action required)

Specify options

This 1 variance analysis requires no further modifications, but other options are available. Use the Options subdialog box to change the confidence level or to specify a one-sided test.
  • (no action required)
