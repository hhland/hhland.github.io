Description of how to perform 1-Sample t using Minitab  

Scenario

Energy expenditure is measured for 15 randomly selected, upper-income families in the United States to see if the mean expenditure differs from the published value of $1080.

Choose the appropriate analysis

To compare a population mean to a target value, use 1-Sample t.
  1. Open the worksheet ENERGY.MTW.
  2. Choose Stat > Basic Statistics > 1-Sample t.

Enter your data

The worksheet contains raw data in a single column.
  1. Choose One or more samples, each in a column and enter 'Energy $'.

Perform hypothesis test

Use this option to test whether the population mean equals a target value.
  1. Check Perform hypothesis test.
  2. In Hypothesized mean, type 1080.

Specify options

Use the Graphs subdialog box to visualize your data with graphs. Use the Options subdialog box to change the confidence level or to request a one-sided hypothesis test.
  1. Click Graphs.
  2. Check Individual value plot.
  3. Click OK in each dialog box.
