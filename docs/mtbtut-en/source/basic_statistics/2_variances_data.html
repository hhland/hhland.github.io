Data requirements for 2 Variances  

What kind of data is required?

You must provide numeric, continuous measurements for both random samples. Data can come from any distribution. For example:
  • Length of beams in a lumber mill
  • Force required to open pretzel bags
  • Volume of soda in bottles

What should my worksheet look like?

Raw data

Raw data can be stacked or unstacked. Here are two ways of entering the same raw data:

To enter stacked raw data, list observations from both samples in one column, and list subscripts to identify the samples in the second column.

Summarized data

To enter summarized data, you do not need data in a worksheet. Simply type values for the sample size and variance (or standard deviation) of each sample into the dialog box.
