Description of how to perform the Display Descriptive statistics analysis using Minitab  

Scenario

An aspiring junior meteorologist collected data about her hometown weather over the course of a year. The values indicate the number of days per month with significant precipitation.

Choose the appropriate analysis

Because this is her first look at these data, reviewing descriptive statistics will give her a broad overview of some of their most important features, such as measures of central tendency and spread.
  1. Open the worksheet PRECIPITATION.MTW.
  2. Choose Stat > Basic Statistics > Display Descriptive Statistics.

Enter the data to be analyzed

You'll get a separate analysis for each variable that you enter in Variables. In this case, there is only one.
  1. In Variables, enter Precipitation.
  2. Click OK.

Specify a by variable, if necessary

A by variable is (generally) a categorical variable that you can use to split an analysis into multiple parts that you can compare and contrast. For example, if you specify gender as a by variable, you get separate results for male and female observations. This analysis, however, has no by variable.

(no action required)

Specify options

This analysis requires no further modifications, but other options are available. Click Statistics to change the default statistics produced in the analysis. Click Graphs to add illustrative graphs of your data.

(no action required)
