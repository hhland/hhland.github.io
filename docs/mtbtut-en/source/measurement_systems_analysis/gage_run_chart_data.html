Data requirements for Gage Run Chart  

What kind of data is required?

Your measurement data must be numeric and continuous. For example:

  • Weight of potatoes in pounds
  • Length of an automotive part in inches
  • Diameter of piston rings in centimeters

You also need a part name or number column and a column for the operators. Parts and operators can be text or numbers. For example:

  • Parts A, B, C, and D and operators 1, 2, 3
  • Parts 111, 117, and 119 and operators Sam and Laura

A Gage Run Chart requires balanced designs (equal numbers of observations per cell) and replicate measurements. Select parts that represent the actual or expected range of process variation.

Use Create Gage R&R Study Worksheet to organize your data collection for correct data analysis. Minitab creates the worksheet based on the number of parts, replicates, and operators.
