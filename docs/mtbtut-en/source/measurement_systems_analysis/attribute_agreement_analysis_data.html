Data requirements for Attribute Agreement Analysis  

What kind of data is required?

You must supply the ratings that are given by each evaluator. These variables must be columns of numeric or text data. They can be binary (Good/Bad), nominal (Blue/Yellow/Red), or ordinal (Poor/Fair/Good).

You must distinguish these ratings according to the evaluator and trial to which they correspond.

If a standard exists, you can enter a Standard column to indicate the "correct" rating for each item.

Use Create Attribute Agreement Analysis Worksheet to organize your data collection for correct data analysis. Minitab creates the worksheet based on the number of appraisers, samples, and replicates.

What should my worksheet look like?

Stacked data

A stacked column of ratings, with corresponding stacked columns to identify their evaluator, subject, and trial. A standard column is optional. This format is created by Create Attribute Agreement Analysis Worksheet.
