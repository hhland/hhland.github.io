Data requirements for Binary Logistic Regression  

What kind of data is required?

Your response data must be binary, with two distinct values (numeric or text). Your data for each predictor variable can be of any type.

What should my worksheet look like?

Binary data

To enter binary data, provide one column of data for your response variable and one column of data for each predictor. A worksheet of binary data has a row for every trial; a trial consists of a combination of predictor values and the corresponding response value.

Frequency data

To enter frequency data, provide a single column of data for your response variable, a column of data for each predictor, and a column of nonnegative numeric frequencies. Do not enter a row of data for every trial; instead, enter a row for each unique combination of response and predictor values and enter a frequency column to indicate how many times you observed each combination.

Summarized data

To enter summarized data, provide a column of data for each predictor, a column of the number of trials that occurred with each combination of predictors, and a column of the number of events that occurred in these trials.
