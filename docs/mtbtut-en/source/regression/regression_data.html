Data requirements for General Regression  

What kind of data is required?

Your response variable must be numeric and continuous. Your predictor variables can be continuous or categorical.

What should my worksheet look like?

Provide one column of data for the response and one column of data for each predictor variable.

All columns must be the same length. Each row in your worksheet represents measurements on one observation or subject.
