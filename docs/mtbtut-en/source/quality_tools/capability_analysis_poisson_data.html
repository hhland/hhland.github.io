Data requirements for Poisson Capability Analysis  

What kind of data is required?

Your data must follow the Poisson distribution. In other words, they must represent counts of events, defects, or activities. Also, the data must be nonnegative integers. For example:
  • Number of delayed flights in one day
  • Number of scratches on a computer screen
  • Number of errors on a standardized survey

You must also enter the sample size: the dimensions of the object that you inspected for defects or the length of time in which you counted events. The sample size can be any positive value.

What should my worksheet look like?

Enter all observations in a single column, with a second column of sample sizes

If all sample sizes are equal, you can omit the column of sample sizes and simply enter the sample size in the dialog box.
