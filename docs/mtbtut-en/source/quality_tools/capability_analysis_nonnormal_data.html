Data requirements for Nonnormal Capability Analysis  

What kind of data is required?

Your measurement data must be numeric and continuous. For example:
  • Customer wait time
  • Fill weight of a cereal box
  • Tensile strength of electric cords

The data can follow any nonnormal distribution.

Control and Stability

The results of a capability analysis are invalid if the process mean and variation are not in statistical control. Use control charts to verify stability.

Enter measurements one subgroup per row

The first row contains observations from the first subgroup, the second row contains observations from the second subgroup, and so on.

Nonnormal capability analysis does not analyze within-subgroup performance. Therefore, if you use this option, Minitab simply stacks the data and treats them as a single sample.
