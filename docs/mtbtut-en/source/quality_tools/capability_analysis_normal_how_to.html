Description of how to perform a Normal Capability Analysis using Minitab  

Scenario

An engine manufacturer uses a forging process to produce piston rings. The forging process is in control, and now the quality engineers want to assess the process capability. They collect twenty-five samples of five piston rings. The specification limits for piston ring diameter are 74.0 Â± 0.05.

Choose the appropriate analysis

The quality engineers want to assess the capability of normal data, so they should use a normal capability analysis.
  1. Open the worksheet PISTON.MTW.
  2. Choose Stat > Quality Tools > Capability Analysis > Normal.

Enter your data

This worksheet has a single column of measurements and a constant subgroup size of five.
  1. In Single column, enter Diameter.
  2. In Subgroup size, type 5.

Set your specifications

You must enter at least one specification limit. The diameter measurements have a lower specification of 73.95 and an upper specification of 74.05.
  1. In Lower spec, type 73.95.
  2. In Upper spec, type 74.05.

Specify options

Many options are available with this capability analysis. To analyze nonnormal data by applying the Johnson or Box-Cox transformations, click Transform. To change the way Minitab estimates within-subgroup variation, click Estimate. Click Storage to store capability statistics. In this case, the process has a target value of 74, so this value is noted in Options.
  1. Click Options.
  2. In Target, enter 74.
  3. Click OK in each dialog box.
