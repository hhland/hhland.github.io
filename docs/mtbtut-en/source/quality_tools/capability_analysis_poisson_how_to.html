Description of how to perform a Poisson Capability Analysis using Minitab  

Scenario

A wire manufacturer is concerned about the effectiveness of the wire insulation process. The quality engineers randomly selected lengths of electrical wire and tested them for weak spots in the insulation by subjecting them to a test voltage. They recorded the number of weak spots (defects) and the length of each wire in meters.

Choose the appropriate analysis

The wire defect counts are discrete data that follow a Poisson distribution, so it is appropriate to use a Poisson capability analysis.
  1. Open the worksheet WIRE.MTW.
  2. Choose Stat > Quality Tools > Capability Analysis > Poisson.

Enter your data

Enter the column that contains defect counts, and specify whether the sample size is constant or variable according to values in a second column. If the process has a historical average, type it here.
  1. In Defects, enter 'Weak Spots'.
  2. Under Sample size, choose Use sizes in and enter Length.
  3. Click OK.

Specify other options

This capability analysis requires no further modifications, but other options are available. Specify a target and a historical value for the average DPU in the main dialog box. Use the subdialog boxes to request different tests for special causes, to specify a title for the analysis, or to store process statistics.
  • (no action required)
