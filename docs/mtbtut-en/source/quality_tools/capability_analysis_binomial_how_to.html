Description of how to perform a Binomial Capability Analysis using Minitab  

Scenario

A call center coordinator wants to assess the capability of the center's call answering process. The total number of incoming calls and non-answered calls (defectives) were recorded for 20 days.

Choose the appropriate analysis

Each call has only two options: either it was answered or not. These values are discrete data that follow a binomial distribution, so a binomial capability analysis is appropriate.
  1. Open the worksheet UNAVAILABLE.MTW.
  2. Choose Stat > Quality Tools > Capability Analysis > Binomial.

Enter your data

Enter the column that contains the number of defectives (unanswered calls), and then specify whether the sample size is constant or whether it varies according to values in a second column. In this example, sample size does vary.
  1. In Defectives, enter Unavailable.
  2. Under Sample Size, choose Use sizes in and enter Calls.
  3. Click OK.

Specify other options

This capability analysis requires no further modifications, but other options are available. In the main dialog box, specify a target proportion of defectives and a historical average proportion. Use the subdialog boxes to request different tests for special causes, to specify a title for the analysis, or to store process statistics.
  • (no action required)
