如何使用 Minitab 创建 C 控制图的说明  

方案

一家墙纸制造商要评估其印刷过程的稳定性。 从 25 卷墙纸中抽取了 100 英尺的样本，并计算了瑕疵数，其中包括印刷污渍、图形趋势失真和缺墨。

选择适当的分析方式

C 控制图和 U 控制图都可显示缺陷数（此处为印刷瑕疵），当样本大小恒定时（此处为 100 英尺长的样本），这两种控制图都适用。 在此例中，出于个人喜好选择了 C 控制图。
  1. 打开工作表“墙纸.MTW”。
  2. 选择统计 > 控制图 > 属性控制图 > C。

输入数据

您具有一列计数；在变量中输入多少列就会生成多少个 C 控制图。
  1. 在变量中，输入墙纸。

指定选项

此控制图可提供多个有用的选项。 您可以用它们来添加有用的标签、将控制图分为几个阶段、为控制图使用的数据创建子集等。 在本例中，您所关心的是不要丢失任何可能的变异特殊原因。
  1. 单击 C 控制图选项，然后选择检验。
  2. 选择执行所有的特殊原因检验，然后在每个对话框中单击确定。

解释输出

现在应该做些什么？ 有关如何解释此分析结果的指导信息，请参见 StatGuide。
