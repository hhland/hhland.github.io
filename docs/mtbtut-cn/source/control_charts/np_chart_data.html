NP 控制图的数据要求  

需要什么类型的数据？

您必须具有一个包含每个子组缺陷数的列。 如果子组大小不是常量，您还必须具有一个包含相应子组大小的列。 如果子组大小是常量，您就无需在工作表中记录它，只需要将这个值标注在对话框中。 例如：
  • 每天没有通过检验的电视机数（缺陷）和被检查的电视机数（子组）
  • 缺少关键信息的急症室登记表数（缺陷）和已提交的登记表总数
  • 递送公司机队的停飞飞机数（缺陷）和该机队所拥有的飞机总数
注 您也可以使用 P 控制图监视缺陷，因为它具有相同的数据要求。 当子组大小不同时 P 控制图可能更适用，因为它保持一条恒定的中心线。
