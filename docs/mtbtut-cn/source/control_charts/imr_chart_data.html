I-MR 控制图的数据要求  

需要什么类型的数据？

每个控制图只能包含一个数字测量值。 根据定义，I-MR 控制图的数据并不会分为多个子组，因为您无法采用这种方式适当地收集这些数据；过程可能会很少发生，或者更频繁地取样可能会很难或者很昂贵。 例如：
  • 花岗岩处理厂的每周废料总量
  • 大型望远镜镜片的抛光时间
  • 喷气发动机的失效速度
  • 模块式房屋的组装时间
注 如果您可以将数据收集到多个子组中，通常这样会更方便。 Xbar-R 控制图和 Xbar-S 控制图与 I-MR 控制图相似，但它们要求使用子组。
