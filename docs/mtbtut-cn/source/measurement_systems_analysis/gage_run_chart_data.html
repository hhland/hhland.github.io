量具运行图的数据要求  

需要什么类型的数据？

测量数据必须为连续的数值型数据。 例如：

  • 马铃薯的重量（磅）
  • 汽车部件的长度（英寸）
  • 活塞环的直径（厘米）

您还需要一个部件名称列或部件号列以及一个操作员列。 部件和操作员可以使用文本或数字表示。 例如：

  • 部件 A、B、C 和 D 以及操作员 1、2、3
  • 部件 111、117 和 119 以及操作员 Sam 和 Laura

量具运行图需要平衡设计（每个单元格的观测值个数相同）和仿行测量值。 请选择表示过程变异的实际或预期极差的部件。

使用创建量具 R&R 研究工作表来组织您所收集的数据，以便正确地分析数据。 Minitab 会基于部件数、仿行数和操作员数来创建工作表。
