用法 1： 确定测量系统变异性的来源

一家撞击试验公司对其近期的测量值进行了审核。 该公司发现其测量值存在严重的变异性问题，但无法确定问题的来源。 如果测量工具和操作员的操作中只有一项导致此问题的发生，该公司不愿在这两方面都花费时间和金钱。

您可以使用量具运行图将您的注意力集中在导致测量系统变异的根本原因上： 测量设备、操作员，或两者。

用法 2： 检测测量数据中的模式

四个检查员检验平板玻璃样本是否有瑕疵。 然而其中一个检查员的拒绝率比其他检查员高很多，他的主管没有注意到这一差异。 经过进一步调查，发现该名检查员误拒了品质好的玻璃。

量具运行图可以帮助您识别出测量值中的不同模式，从而提供辨明测量设备或操作员相关问题的实质的能力。
