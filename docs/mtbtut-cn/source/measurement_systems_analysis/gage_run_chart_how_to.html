如何使用 Minitab 创建量具运行图的说明  

方案

选择了十个表示过程变异预期极差的部件。 由三名操作员按照随机顺序测量十个部件的厚度，每个部件测量两次。

选择适当的分析方式

因为您希望绘制测量系统的分析结果图，所以，量具运行图适用。
  1. 打开工作表“注塑.MTW”。
  2. 选择统计 > 质量工具 > 量具研究 > 量具运行图。

输入数据

部件号、操作员和测量数据都是必需的。
  1. 在部件号中，输入部件。
  2. 在操作员中，输入操作员。
  3. 在测量数据中，输入响应。
  4. 单击确定。

指定选项

此运行图无需进一步修改，但您可以选择其他选项。 单击量具信息以指定量具研究的信息或者单击选项以指定运行图的标题。
  • （无需任何操作）

解释输出

现在应该做些什么？ 有关如何解释此分析结果的指导信息，请参见 StatGuide。
