属性一致性研究的数据要求  

需要什么类型的数据？

您必须提供每个评估者的评级。 这些变量必须是数字列或文本数据列。 它们可以是二值变量（好/坏）、名义性变量（蓝色/黄色/红色）或顺序变量（差/中/好）。

您必须根据评估者以及它们所对应的试验来区分这些评级。

如果有标准，您可以输入一个“标准”列来指示每项的“正确”评级。

请使用创建属性一致性研究工作表来组织您所收集的数据，以便正确地分析数据。 Minitab 基于评检验员、样本数和仿行数来创建工作表。
