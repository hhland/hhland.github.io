用途 1： 使设计空间可视化

食品科研人员要进行一次混料试验，以确定将生产出最佳口味的奶油软糖所需的糖、可可粉、黄油和香精的量。 他们创建了试验设计，现在想直观地查看该设计。

单纯形设计图

单纯形设计图可以显示试验中使用的设计点。 这些设计点用于定义在每次试验运行中使用的不同分量的比率（或量）。

用途 2： 评估响应与过程变量如何相关

一家油漆制造商要进行一次混料试验，以使其油漆的粘结性能最大化。 研究人员要考虑两个过程变量： 涂漆表面和涂漆时的温度。

过程变量是试验中并非混料一部分，但却会影响混料混合属性的因子。 使用因子图可评估这些过程变量对响应产生怎样的影响。

主效应图

显示每个过程变量各水平的均值。
交互作用图

显示两个因子的每种水平组合的均值响应。
立方图

显示最多八个因子的每种因子水平组合的均值响应。

用途 3： 评估分量影响响应的程度

农业科研人员要进行一次混料试验，以确定各种气候条件下哪种比率的氮、磷酸和钾可促进草的生长。 他们想创建一个图形来向产品经理解释这些分量效应。

响应跟踪图

显示分量更改如何对响应产生影响（相对于参考混合）。 您可以使用响应跟踪图来确认影响最大的分量。 如果设计包含过程或数量变量，则它们必须保持在固定水平上。

用途 4： 找出最佳配方

一家制药公司希望找到要在止咳糖浆配方中使用的四种成分的最有效剂量。

等值线图
曲面图
使用等值线图和曲面图可以找出提供所需响应的分量设置。 这两个图将显示其它常量保持不变的情况下，某些分量如何对响应变量产生影响。 要获得更全面的解释，请同时检查等值线图和曲面图。
注 此外，还有一个关于响应优化器的教程，它是一种可以帮助确定优化单个响应或一组响应的分量配方的工具。
