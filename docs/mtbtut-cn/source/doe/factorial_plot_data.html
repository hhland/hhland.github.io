我的工作表的外观如何？

当您创建设计时，Minitab 将在不同列中存储标准序、运行序、中心点指示符、区组分配和每个因子。 工作表中的每一行都包含与一次试验运行对应的数据。 在您进行试验之后，请输入每次试验运行的响应数据。 工作表的确切组成将根据所选的设计、因子信息和响应而有所不同。
