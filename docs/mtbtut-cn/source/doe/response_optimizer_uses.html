用途 1： 找出可产生最佳结果的设置或配方

一家家用电器制造商正要研制一种新的洗衣机产品线，并想找出可洗出最干净衣服的水温、搅拌转速和洗衣时间周期。

在您确定可显著影响响应的因子或分量之后，Minitab 的响应优化器可帮助您确定获取所需结果的正确设置。 您可以对因子、响应曲面或混料试验设计使用响应优化器。

用途 2： 优化多个响应

一位工业工程师想找到制造塑料包装材料的理想合成聚合物、处理工艺和处理温度。 其目标是找出即可最小化生产时间，又可最大化材料断裂强度的设置。

通常，您必须在设计的试验中分析多个重要的响应变量。 但是，一个响应的最优设置可能与另一个响应的最优设置差别很大。 使用 Minitab 的响应优化器可找出所有重要响应的折衷设置。

用途 3： 最小化成本并获得好的结果

前几个星期，一家鞋制造厂生产了大量有缺陷的合成皮革靴子。 质量经理设计了一个用于检验导致成品缺陷的填充物水平、固化剂和应用方法的效应的试验。 除了减少缺陷之外，该制造厂还希望降低生产成本。

许多时候，可产生最佳结果的因子设置其运行成本也是最昂贵的。 通过在分析中考虑每种处理组合的成本，即可在最小化成本和优化响应之间找到一个折衷设置。
