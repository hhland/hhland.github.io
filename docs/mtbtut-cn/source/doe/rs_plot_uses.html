用途 1： 找出使用等值线图的最佳操作条件

一家谷类食品公司要进行一次响应曲面试验，以确定生产口味最佳的谷类食品所需的糖和蜜的量。 首席食品科研人员想要在等值线图中显示结果。

等值线图

用于找出提供所需响应的因子水平设置。 等值线图提供了一个二维视图，在该视图中，可产生相同响应的设置显示为恒定响应的等值线。 要获得更全面的解释，请结合使用曲面图和等值线图。

用途 2： 找出使用曲面图的最佳操作条件

研究人员研制了一种新的洗衣粉，现在想知道哪种水温、循环时间和洗衣粉浓度设置可以洗涤出最光洁的布料。 他们进行了一次响应曲面试验，用于确定哪些因子水平或因子水平组合可使光洁度最大化。 研究人员想用一个图形来向产品开发团队展示最佳因子设置。

曲面图

用于找出提供所需响应的因子水平设置。 曲面图可提供一个三维视图来显示这些因子如何对响应产生影响。 要获得更全面的解释，请结合使用曲面图和等值线图。
注 此外，还有一个关于响应优化器的教程，它是一种可以帮助确认共同优化单个响应或一组响应的输入变量设置组合的类似工具。
