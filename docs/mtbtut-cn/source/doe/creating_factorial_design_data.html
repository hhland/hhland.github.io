的数据要求  

需要什么类型的数据？

使用创建因子设计不需要任何数据。 只需定义一个设计并输入因子的相关信息，Minitab 将汇集需要在分析因子设计中使用的数据（响应数据除外，您必须收集响应数据）。

设计类型

试验中的因子数、每个因子中的水平数和其他考虑因素将决定适合您所处情况的设计类型。 Minitab 提供二水平、Plackett-Burman 和一般全因子设计，您可以自定义这些设计以满足试验需要。

因子信息

您至少必须具有两个因子，每个因子至少必须有两个水平（或者，对于一般全因子设计，水平数可能会更多）。 因子可以是类别因子或连续因子。 因子设置可以为文本（如高和低）或数字（如 100°和 200°）。

其他考虑因素

试验的目的可能是要决定需要更高还是更低的统计功效（例如，非常敏感的最终评估与早期筛选分析）。 有限的预算可能也会限制您能够执行的试验运行总次数。 通过指定不同数量的中心点和仿行次数，您可以对这些方面进行一定程度上的控制。 Minitab 可提供 2 水平因子、Plackett-Burman 和一般全因子设计的功效和样本数量评估，从而帮助您选择适合于您的实际情况的设计。

响应数据

在您进行试验之后，请将每次试验运行的响应数据输入到工作表中。 最多可有 25 个数值型响应数据列。 例如，响应变量可为：
  • 化学反应的产量
  • 处理订单的时间
  • 塑料强度

我的工作表的外观如何？

当您创建因子设计时，Minitab 将在不同列中存储标准序、运行序、中心点指示符、区组分配和每个因子。 工作表中的每一行都包含与一次试验运行对应的数据。 工作表的确切组成将根据所选的设计和因子信息而有所不同。
