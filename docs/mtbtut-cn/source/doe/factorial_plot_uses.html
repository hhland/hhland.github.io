用途 1： 显示一个或多个因子的所有水平的均值响应

一家网上鞋店的质量经理进行了一个设计试验来确定哪些因子会对处理订单所需要的时间产生影响。 她确定了三个具有显著统计意义的因子，并希望对所得结果绘图以供管理团队参考。

使用因子图可以直观地显示因子如何对响应产生影响。

主效应图

显示每个因子各个水平的均值响应。
交互作用图

显示两个因子的每种水平组合的均值响应。
立方图

显示最多八个因子的每种因子水平组合的均值响应。

用途 2： 找出最佳操作条件

一位工业工程师想找到可使塑料包装材料断裂强度最大化的合成聚合物、处理工艺和处理温度的设置。

等值线图
曲面图
使用等值线图和曲面图可以找出提供所需响应的因子水平设置。 这两个图将显示其它常量保持不变的情况下，某些因子如何对响应变量产生影响。 要获得更全面的解释，请同时检查等值线图和曲面图。
注 此外，还有一个关于响应优化器的教程，该优化器是一种可以帮助确认共同优化单个响应或一组响应的输入变量设置组合的工具。
