需要什么类型的数据？

使用创建响应曲面设计不需要任何数据。 只需定义一个设计并输入因子的相关信息，Minitab 将汇集需要在分析响应曲面设计中使用的数据（响应数据除外，您必须收集响应数据）。

设计类型

试验的目标、可执行的试验运行的预算或实际限制以及其他考虑因素将决定适合您所处情况的设计类型。 Minitab 提供中心复合和 Box-Behnken 设计，您可以自定义这两种设计来满足您的试验需求。

因子信息

您可以具有两个到十个因子。 因子可以是类别因子或连续因子。 因子设置可以为文本（如高和低）或数字（如 100°和 200°）。

其他考虑因素

试验中设定的各种限制和要求都有可能对设计产生影响。 您可以指定将影响设计中试验运行的次数和组成的中心点和仿行。

响应数据

在您进行试验之后，请将每次试验运行的响应数据输入到工作表中。 最多可有 25 个数值型响应数据列。 例如，响应变量可为：
  • 化学反应的产量
  • 处理订单的时间
  • 塑料强度

我的工作表的外观如何？

当您创建设计时，Minitab 将在不同列中存储标准序、运行序、点类型、区组分配以及每个因子。 工作表中的每一行都包含与一次试验运行对应的数据。

如果您指定了因子水平设置，Minitab 将会在数据表中显示实际值。 否则，Minitab 将采用已编码单位显示因子水平： -1 表示低值，0 表示中心点，1 表示高值。 轴点被编码为 -1.41421 和 1.41421。 在任何一种情况下，您通常都希望按已编码单位分析您的设计。
