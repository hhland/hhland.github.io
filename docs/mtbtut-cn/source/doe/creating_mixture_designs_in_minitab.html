  • 在 Minitab 中创建混料设计

有关如何使用 Minitab 的说明  

情形

一家食品实验室的研究人员正在尝试改进酒味乳酪的烹饪方法。 他们要研究混料混合和火锅的操作温度如何影响口味。

混料由三种成分组成： 两种类型的瑞士奶酪（Emmenthaler 和 Gruyere）和白葡萄酒。

选择适当的设计和分量数

研究人员想要进行一个具有 3 个分量（两种奶酪和酒）和一个过程变量（操作温度）的极端顶点设计。
  1. 选择统计 > DOE > 混料 > 创建混料设计。
  2. 在设计类型下，选择极端顶点。
  3. 从分量数中，选择 3。
  4. 单击设计。 单击确定。

命名分量并定义上下限

为分量指定上下限。 在本示例中，已根据过去的经验确定了整个火锅配方中每种成分的限制。
  1. 单击分量并按照以下方式填充表格：
    分量 名称 下限 上限
    A Emmenthaler 奶酪 0.2 0.6
    B Gruyere 奶酪 0 0.3
    C 酒 0.4 0.6
  2. 单击确定。

定义过程变量

您最多可以定义 7 个过程变量，尽管本示例中仅使用了一个过程变量： 操作温度。 您必须指定过程变量的名称、数据类型以及低设置和高设置。
  1. 单击过程变量。
  2. 在过程变量下，选择数字并按照以下方式填充表格：
    过程变量 名称 类型 低 高
    X1 温度 数字 80 90
  3. 在每个对话框中单击确定。
