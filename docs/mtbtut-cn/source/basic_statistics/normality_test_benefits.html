用法 1： 确定您的数据是否遵循正态分布

一名研究人员希望研究喷气发动机的某个关键部件的铸造过程。 由于需要使用正态分布数据（如，能力分析）的检验在与非正态数据结合使用时会产生误导性结果，因此，该研究人员需要确保数据遵循正态分布。

通常，统计过程都会假定数据来自正态分布。 尽管许多检验不使用正态数据也可以正常工作，但在某些情况下，使用非正态数据会产生误导性结果。 请使用正态性检验来确定数据是否遵循正态分布。
