如何使用 Minitab 执行 Poisson 能力分析的说明  

方案

一家电线制造商关心电线绝缘工序的有效性。 质量工程师随机选择电线的长度，并通过对电线施加一定的检测电压来测试电线中的绝缘弱点。 他们记录了弱点（缺陷）的数量以及每根电线的长度（单位为米）。

选择适当的分析方式

电线缺陷数为服从 Poisson 分布的离散数据，因此 Poisson 过程能力分析适用。
  1. 打开工作表“电线.MTW”。
  2. 选择统计 > 质量工具 > 能力分析 > Poisson。

输入数据

输入包含缺陷数的列，然后根据第二列中的值指定样本大小为常量还是变量。 如果过程具有历史平均值，请在此输入。
  1. 在缺陷中，输入'弱点'。
  2. 在样本大小下，选择实际样本量并输入长度。
  3. 单击确定。

指定其他选项

此能力分析无需进一步修改，但您可以选择其他选项。 在主对话框中指定平均 DPU 的目标值和历史值。 使用子对话框针对不同的特殊原因请求不同的检验、指定分析标题或者存储过程统计量。
  • （无需任何操作）

解释输出

现在应该做些什么？ 有关如何解释此分析结果的指导信息，请参见 StatGuide。
