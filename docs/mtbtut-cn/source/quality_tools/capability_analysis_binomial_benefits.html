用法 1： 执行包含二项数据的能力分析

一家呼叫中心需要满足其客户对正确拨打的销售电话比率的规定。 尽管可以使用能力分析，但呼叫中心数据为二项数据： 即电话已正确拨打或并未正确拨打。使用常规能力工具可能会产生误导性结果。

如果您希望执行能力分析且具有二项数据，请使用二项能力分析。

用法 2： 确定缺陷品的比率是否符合您的规定要求

为了保持合同有效，一家汽车供应商生产的缺陷火花塞不得超过的最大比率。 不符合规格的产品是不能接受的，这意味着供应商遭到损失。

您可以使用过程能力分析对您所希望的过程水平（规格限）和过程的实际水平进行比较。
