用法 1： 执行包含 Poisson 数据的能力分析

一家纺织品制造商需要符合客户对每码织物缺陷数的规格要求。 因为该制造商要测量缺陷率，所以，这些数据遵循 Poisson 分布。 尽管可以进行过程能力分析，但正态过程能力分析工具可能会产生误导性结果。

如果您希望执行能力分析且具有 Poisson 数据，请使用 Poisson 能力分析。

用法 2： 确定过程的缺陷率是否符合规格要求

一家呼叫中心的次级承包商需要符合其客户对成功呼叫的要求。 无法带来销售量的高呼叫率可能会意味着丢失此承包合同。

您可以使用过程能力分析对您所希望的过程水平（规格限）和过程的实际水平进行比较。
