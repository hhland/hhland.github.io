用法 1： 执行包含非正态数据的能力分析

一个造船厂的工头需要满足政府对船体上大型焊接提出的强度规格。 尽管可以使用能力分析，但焊缝强度数据并非正态分布： 使用正态能力工具可能会产生误导性结果。

如果您希望执行能力分析且具有非正态数据，请使用非正态能力分析。

用法 2： 确定过程的非正态输出是否满足规格要求

一家医用乳胶手套的制造商尝试满足外科协会指定的目标厚度。 虽然手套的平均厚度与目标厚度非常接近，但与可接受厚度相比，大量手套要么更厚一点，要么更薄一点。

您可以使用过程能力分析对您所希望的过程水平（规格限）和过程的实际水平进行比较。
