非正态能力分析的数据要求  

需要什么类型的数据？

测量数据必须为连续的数值型数据。 例如：
  • 客户等候时间
  • 谷类食品盒的填充重量
  • 电源线的拉伸强度

数据可以遵循任意非正态分布。

控制和稳定性

如果过程均值和过程变异不处于统计受控状态，则能力分析的结果无效。 请使用控制图来验证稳定性。

第一行包含第一个子组的观测值，第二行包含第二个子组的观测值，依此类推。

非正态能力分析并不分析子组内性能。 因此，如果您使用此选项，Minitab 将会堆叠数据并将它们视为一个样本。
