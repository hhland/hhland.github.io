Poisson 能力分析的数据要求  

需要什么类型的数据？

数据必须遵循 Poisson 分布。 也就是说，它们必须表示事件数、缺陷数或活动数。 此外，数据必须为非负整数。 例如：
  • 一天中的延误航班数
  • 计算机屏幕上的划痕数
  • 标准调查中的错误数

您还必须输入样本大小： 您要检测缺陷的对象维度或对事件计数的时间长度。 样本大小可以为任意正数值。

我的工作表的外观如何？

在一列中输入所有观测值，在另一列中输入样本大小

如果所有样本大小都相等，您可以忽略样本大小列，只要在对话框中输入样本大小。
