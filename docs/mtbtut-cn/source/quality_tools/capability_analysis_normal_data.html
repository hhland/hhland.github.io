正态能力分析的数据要求  

需要什么类型的数据？

测量数据必须为连续的数值型数据。 例如：
  • 发动机凸轮轴的直径（厘米）
  • 果汁瓶的容量（毫升）
  • 风扇叶片的长度（英寸）

数据应遵循正态分布，尽管您可以通过应用变换分析非正态数据。

子组

尽可能标示出合理的子组——即具有共同变异源（相同的时间间隔、相同的方法等）的输出批次。 示例包括：
  • 从每一箱中选择的十个灯泡。
  • 每小时监控的六个来电。

将来自不同子组的测量值相混合可能会隐藏过程潜在的重要信息。

某些数据可能没有子组，如破坏性测试的结果，或者每天一次的测量结果。 对于这些情况，请输入子组大小 1。

控制和稳定性

如果过程均值和过程变异不处于统计受控状态，则能力分析的结果无效。 请使用控制图来验证稳定性。

我的工作表的外观如何？

堆叠测量值位于一列中，并且使用另一列指定每个测量值的子组

如果所有子组大小都相等，您可以忽略下标 ID 列并在对话框中输入子组大小。

每行输入一个子组的测量值

第一行包含第一个子组的观测值，第二行包含第二个子组的观测值，依此类推。
