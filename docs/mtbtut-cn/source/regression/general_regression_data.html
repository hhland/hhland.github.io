一般回归的数据要求  

需要什么类型的数据？

响应数据必须是数字和连续数据。 预测变量可以是连续变量或类别变量。

我的工作表的外观如何？

提供一列响应数据，以及一列表示每个预测变量的数据。

所有列的长度必须相同。 工作表中的每一行代表一个观测值或对象的测量值。
