二元 Logistic 回归的数据要求  

需要什么类型的数据？

您的响应数据必须是具有两个可区分值（数字或文本）的二元数据。 您的每个预测变量的数据都可以是任何类型的数据。

我的工作表的外观如何？

原始数据

要输入原始数据，请为响应变量提供一列数据并为每个预测变量提供一列数据。 原始数据工作表中具有一行表示每个试验的数据；一个试验由多个预测变量值及其相应的响应值组成。
C1 C2 C3 C4
订阅 儿童 宠物 私房屋主
1 是 是 是 是
2 是 是 否 是
3 否 否 否 否
4 是 是 否 是
5 否 是 是 是
6 是 是 否 是

频数数据

要输入频数数据，请提供一列响应变量数据、每个预测变量一列数据以及一列非负数的频数数据。 不要输入一行表示每个试验的数据；而是输入一行表示每个响应值和预测变量值的唯一组合的数据，并输入一个表示您观测每个组合的次数的频数列。
C1 C2 C3 C4 C5
订阅者 儿童 宠物 私房屋主 频数
1 是 是 是 是 1
2 是 是 否 是 3
3 否 否 否 否 1
4 否 是 是 是 1

汇总数据

要输入汇总数据，请提供一列表示每个预测变量的数据、一列对每个预测变量组合所发生的试验数的数据以及一列在这些试验中发生的事件数的数据。
C1 C2 C3 C4 C5
儿童 宠物 私房屋主 试验 事件
1 是 是 是 2 1
2 是 否 是 3 3
3 否 否 否 1 0
