如何使用 Minitab 执行二元 Logistic 回归的说明  

方案

一家谷类食品公司正在调查一种名为 Cocoa Crunch 的新产品的电视广告效应。 广告在某一社区播出一周后，他们在一家当地超市的出口处随机访问了七十一位成年人作为样本。

调查问题是：
  • 是否购买了 Cocoa Crunch（购买）
  • 家庭收入（以千美元为单位）（收入）
  • 是否有子女（儿童）
  • 是否观看过广告（收看广告）

选择适当的分析方式

公司希望使用多种预测变量为二元结果建模，因此他们使用二元 Logistic 回归。
  1. 打开工作表“谷类广告.MTW”。
  2. 选择统计 > 回归 > 二元 Logistic 回归。

输入响应变量

公司具有一个二元响应变量的原始数据列。
  1. 选择以响应/频率形式的响应。
  2. 在响应中，输入购买。

输入模型项

在模型中输入所有项，并在因子中输入所有类别变量。
  1. 在模型中，输入收入 儿童 收看广告。
  2. 在因子中，输入儿童 收看广告。
  3. 单击确定。

指定选项

此回归分析无需进一步修改，但您可以选择其他选项。 要创建各种诊断图，请单击图形。 要更改链接功能或进行其他分析修改，请单击选项。 要控制结果的显示方式，请单击结果。 要存储各种统计量，请单击存储。
  • （无需任何操作）

解释输出

现在应该做些什么？ 有关如何解释此分析结果的指导信息，请参见 StatGuide。
