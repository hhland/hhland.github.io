用法 1： 解释预测变量与二元响应变量之间的关系

为了帮助改进广告宣传活动，一家电影工作室希望基于两个预测变量预测人们对观看一部新电影的决定： 年龄和性别。 与其他回归形式不同，其中响应是连续变量（利润、超限量等），二元回归只有两个可能的响应值： 人们观看此电影，或者不看。

请在具有二元响应变量时使用二元 Logistic 回归： 二元响应变量是只有两个可能结果的变量。 例如，您具有一个有缺陷或无缺陷的产品。 二元 Logistic 回归使用来自各种预测变量的数据来为这两个可能结果的概率分别建模。

用法 2： 了解二元结果的几率如何在不同情况下发生变化

一家信用卡公司使用二元 Logistic 回归基于两个预测变量来估计客户延迟支付的几率： 他们的帐户余额以及他们的房子是自己购买的还是租赁的。 该公司可能会对具有较高几率延迟支付的客户指定更高的利息。

两种水平的二元结果（通过/失败、是/否、好/坏）都将出现的可能性可以使用一个称为“优势比”的统计量来表示。 优势比可以帮助您比较不同水平的预测变量对二元结果的效应。 例如，女性（预测变量）购买电动牙刷的可能性（二元结果）是男性的两倍（优势比）。
