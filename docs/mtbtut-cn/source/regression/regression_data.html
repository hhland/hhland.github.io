回归的数据要求  

需要什么类型的数据？

您的预测变量和响应变量必须是数值型的。 例如：
  • 平均户外温度、绝缘水平以及供暖燃料用量
  • 本地收入中位数和本地房价中位数
  • 身高、体重、年龄和血压

我的工作表的外观如何？

提供一列响应变量数据，以及一列表示每个预测变量的数据。

所有列的长度必须相同。
