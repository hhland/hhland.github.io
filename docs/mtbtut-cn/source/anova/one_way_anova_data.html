单因子方差分析的数据要求  

需要什么类型的数据？

必须具有分为两个或多个组的连续数据。 例如：
  • 四种不同品牌的电线样品的强度测量值。
  • 六个不同位置的连锁旅店的客户满意度。
  • 三条不同生产线上生产出来的部件的长度。
这些数据可以采用以下方式进行整理：
  • 一个响应数据堆叠列，具有一个标识各组的对应列
  • 非堆叠响应数据，其中，每组都位于单独的工作表列中

您的样本必须是独立的，即一个样本中的观测值与另一个样本的观测值无关。 与 t 检验不同，方差分析可以比较两组以上的数据。

我的工作表的外观如何？

堆叠数据
