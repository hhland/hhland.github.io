广义线性模型方差分析的数据要求  

需要什么类型的数据？

必须具有一个连续响应变量以及一个或多个类别或连续预测变量。 类别预测变量也称为“因子”，连续预测变量也称为“协变量”。 例如：
  • 一个钢强度值及其工厂位置、班次和钢水温度值
  • 一个刹车距离预测值及其刹车片类型、汽车型号和轮胎型号
  • 一个每天冰激凌销售总量及其日期、月份和最高温度

每次最多可以分析 50 个响应变量、31 个因子以及 50 个协变量。

广义线性模型 (GLM) 在进行方差分析时放宽了对数据的要求。 因子既可以是交叉或嵌套的，也可以是固定或随机的。 协变量可以彼此交叉，也可以与因子交叉，或者是嵌套在因子内。

我的工作表的外观如何？

将您的工作表设置为每个响应变量、因子和协变量均对应一列。 工作表中的每一行代表一个观测值或对象的测量值。 如果某行包含缺失数据，Minitab 将会从所有计算中排除这一整行。
